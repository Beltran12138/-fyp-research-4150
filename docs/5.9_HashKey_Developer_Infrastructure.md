## 5.9.3 hkx Agent Registry — Know Your Agent (KYA)

### The Problem

AI agents now execute 58% of global crypto trading volume (FinanceMagnates, 2026). MiCA's full enforcement deadline lands 1 July 2026 — yet the regulation contains no provision for an autonomous agent as a "client" of a crypto-asset service provider, and no mechanism for agent-level KYC (CryptoNews, 2026). The result is structural: every exchange that allows agent access is operating with an identity gap. The agent trades; no one is accountable.

Coinbase launched Agentic.market on 21 April 2026 — an app store for AI agent services. VALR (South Africa) launched its AI Service suite on 10 April 2026 using the Agent Skills Standard. Neither operates within a KYA framework. Neither is SFC-licensed. The identity problem remains unsolved in every jurisdiction.

### The Concept

**hkx Agent Registry** is HashKey's answer: the first SFC-licensed exchange registry where every AI agent has a verifiable identity, bound to a human principal through a Know Your Agent (KYA) credential layer.

The model is principal-linked accountability:

```
Human Principal (KYC verified)
    → SBT Credential (on-chain identity)
    → Agent Registration (linked to principal)
    → SkillHub Skills (SFC-reviewed)
    → HashKey Trading (regulated infrastructure)
```

Every agent action is attributable to its human principal. Regulatory accountability follows the agent, not just the wallet.
